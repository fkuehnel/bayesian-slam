# experiments/data/

This directory holds **transitory CSV files** generated by the Rust examples.
Its contents are not checked into version control (see the root `.gitignore`).

## How files get here

```bash
cd rust
cargo run --release --example bias_experiment   # writes bias_experiment.csv
cd ..
```

Each Rust example writes its output CSV into this directory.
The Python plotting scripts under `experiments/` then read from here
and produce PGF/PNG figures in `paper/figures/`.

## Why the files are not committed

The CSVs can be large and are fully reproducible from the Rust source,
so they are git-ignored. Regenerate them locally whenever you need
to re-run the plots.
